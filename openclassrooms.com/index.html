Páginas recomendades por GBSoft

Continuamente visitamos páginas Web que querríamos ver en próximos días o simplemente guardarlas para una consulta posterior.
Para esto existen varias herramientas que le permiten guardar un listado de páginas visitadas pero con el inconveniente de tener
que registrarse y sin la posibilidad de agregarlas a una lista personal. Además, tiene una breve descripción y un detalle personal
sobre la página en particular.
Esta página permite (por ahora), consultar las páginas recomendadas o guardadas por mí o mis amigos en la web y allí mismo la
puedo consultar desde cualquier equipo y anónimamente. Debería ser una pestaña fija en mi navegador.
